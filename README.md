# Cisco Multi-VLAN Enterprise Network Implementation

A technical demonstration of building a segmented enterprise network using VLANs and Inter-VLAN routing (Router-on-a-Stick).

## 🛠 Project Overview
This project simulates a corporate network deployment for "TechNova Ltd," focusing on network security through segmentation. By isolating departments into distinct VLANs and managing traffic through a central router, the design ensures improved performance and reduced attack surfaces.

## 🚀 Key Features
- **Network Segmentation:** Implementation of VLANs for traffic isolation.
- **Inter-VLAN Routing:** Configuration of a 2911 Router using sub-interfaces and 802.1Q encapsulation.
- **Switching Logic:** Configuration of Access and Trunk ports on Cisco 2960 switches.
- **IP Management:** Systematic subnetting and gateway assignment for enterprise scalability.

## 💻 Tech Stack
- **Simulation Tool:** Cisco Packet Tracer
- **Hardware Modeled:** Cisco 2911 Router, Cisco 2960 Switches
- **Protocols:** IEEE 802.1Q (Trunking), ICMP (Testing), IPv4, IPv6, OSPF, DHCP

## 📊 Infrastructure Highlights
- **Topology:**Hierarchical with a centralized Router-on-a-Stick.
